## Common Scenarios

### If Status is "Sent" but Email Not Received:
- ✅ Check spam/junk folder
- ✅ Verify recipient email address is correct
- ✅ Check if domain is verified in Resend (if using custom domain)
- ✅ Wait a few minutes (delivery can be delayed)

### If Status is "Failed":
- Check the **Error Message** field for specific error
- Common errors:
  - "No email service configured" → Add `RESEND_API_KEY` to Convex
  - "Domain not verified" → Verify domain in Resend or use `onboarding@resend.dev`
  - "Invalid API key" → Check `RESEND_API_KEY` is correct

### If Status is "Simulated":
- Email was logged but not actually sent
- No email service was configured
- Add `RESEND_API_KEY` to Convex Dashboard → Settings → Environment Variables
- Run `npx convex deploy`
- Try test again
